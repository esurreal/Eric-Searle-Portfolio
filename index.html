Projects

Radical Website

This is a website built in Webflow then exported to VS Code for final adjustments. The client wanted to host it on Netlify. I had fun with this one. It was built for the client's personal Vlog/Blog. This acts as the central hub to all of his web based projects.  

Abella Website

This is a website built for an Italian Restaurant in Atlanta Georgia. It incorporates HTML, Javascript, and CSS. It was also built in Webflow and then exported to VS Code for further adjustments before being pushed to Github Pages for deployment. I used a good mix of stylish typography to try and make it feel elegant, but also warm. I incorporated many interactions to make navigation easier and bring in subtle visual stimulation.

Palaces Website

This is a website I built for a band in Atlanta. They wanted to have some motion in the hero section. I took one of their live videos, compressed it, and used some digital effects on it to make it black and white and grainy. I also incorporated a contact page and email subscription links. Both were done in VS Code after exporting the site off of Webflow. The client was thrilled with the end results.
